Initiative_machine_learning
=========================

项目介绍
------------

Initiative_machine_learning是一个基于python语言开发的项目, 它是一个tutorial目, 旨在记录machine learning过程中学习的各种算法原理和examples.
该项目正在维护中, 不间断更新新的内容.
