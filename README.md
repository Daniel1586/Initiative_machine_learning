项目介绍
------------

Initiative_machine_learning是一个基于python语言开发的项目. 它是一个tutorial项目, 旨在记录工作和学习中遇到的各种machine learning算法, 总结算法原理和开发算法examples.
该项目正在维护中, 不间断更新新的内容.
